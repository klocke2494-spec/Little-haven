# Little Haven

Clean your home. Grow your haven.

Little Haven is a cozy cleaning motivation app with:
- Maple the highland cow companion
- Daily cleaning quests
- Energy modes, including low-energy and flare-day options
- XP, coins, levels, and streaks
- A tiny haven you can decorate
- Custom cleaning quests

## How to run
Open `index.html` in a browser.

## How to put on iPhone
After uploading to GitHub Pages:
1. Open the GitHub Pages link in Safari.
2. Tap Share.
3. Tap Add to Home Screen.
